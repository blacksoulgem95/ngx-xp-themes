NGX XP Icons

An Angular Library for Vintage Icons

Installation

  1. Install the package: npm i --save ngx-xp-icons
  2. Add XP Icons assets to your angular.json 
    
"assets": [
  ...,
  {{ '{' }}
    "glob": "**/*.png",
    "input": "node_modules/ngx-xp-icons"
  {{ '}' }}
],
